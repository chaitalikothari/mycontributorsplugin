This Plugin adds contributors metabox on post page in dashboard. Checked contributors are displayed on front end single post page.

== Description ==

1) This Plugin adds contributors metabox on posts in dashboard.

2) It displays the list of users registered in site.

3) Checked users are displayed on the posts single page alongwith their gravatars.

4) When clicked on user it gets reirected to particular author's page.

== Installation ==

1) Upload the entire slideshow folder to the /wp-content/plugins/ directory.

2) Activate the plugin through the 'Plugins' menu in WordPress.

You will find Contributors metabox on posts in dashboard.
